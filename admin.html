Importar Lecturas de Micromedidores

Sube un archivo en formato Excel (.xlsx, .xls) o CSV (.csv) para registrar múltiples lecturas de contadores de forma masiva.

Instrucciones:

  1. Descargue la plantilla para asegurarse de que su archivo tiene el formato correcto.
  2. Llene la plantilla con los datos de las lecturas. No cambie el nombre de las columnas.
  3. Las columnas requeridas son: NUID, Lectura_Actual (en m³), y Fecha_Lectura (AAAA-MM-DD).
  4. Seleccione el archivo completado y haga clic en "Procesar Archivo".
Descargar Plantilla
Ningún archivo seleccionado
Solo se permiten archivos con extensión .csv, .xls o .xlsx

Pasos para importar pagos

  1. Descargar el formato dando clic en el siguiente botón:
    Descargar Excel
  2. Ingresar la información correspondiente a cada suscriptor, la fecha de pago debe estar en el formato dd/mm/yyyy.
  3. El campo "DESCRIPCIÓN" es opcional.
  4. Los valores válidos para el campo "MÉTODO DE PAGO" son los siguientes:
    • Efectivo
    • Transferencia
  5. Una vez completada la información, subir el archivo utilizando el formulario que encontrará a continuación.
  6. Si el pago no aplica para un suscriptor dado, deje el valor en cero, también puede eliminar la fila completamente.

Pasos para importar otros cobros

  1. Descargar el formato csv En este enlace.
  2. Ingresar la información correspondiente a cada suscriptor.
  3. Una vez completada la información, subir el archivo utilizando el formulario que encontrará a continuación.

No deje valores vacíos. Si el pago no aplica para un suscriptor dado, deje el valor en cero o elimine la fila completamente.
